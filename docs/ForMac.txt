-  教材ファイルの入手

    % cd
    % cd work
    % git clone https://github.com/rmaezono/qmcTutorials.git

    として教材ファイル一式を入手。

    以降、適宜、
    ~/work/qmcTutorials/docs/ForMac.txt 
    を参照。

-  必要な環境のインストール

    % brew install open-mpi
    % brew install emacs
    % brew install gnuplot
    % brew install grace
    % brew install --cask xquartz

-  .bash_profileファイルへの追記

    % cd
    % echo 'export CASINO_ARCH=macos-gnu-parallel' >> .bash_profile
    % echo 'PATH="$PATH":$HOME/work/qmcTutorials/casino/[改行しないで次行に繋げる]
        bin_qmc/$CASINO_ARCH/opt' >> .bash_profile
    % echo 'PATH="$PATH":$HOME/work/qmcTutorials/casino/bin_qmc' >> .bash_profile
    % echo 'export PLOTTER=/opt/homebrew/bin/xmgrace' >> .bash_profile

    を実施。

    % cd
    % cat .bash_profile

    として上記が追記されているか確認。
    
    % exit

    として再度ターミナルを立ち上げる。
    
-  CASINOのインストール

    % cd
    % cd work/qmcTutorials/casino

    % make -j 4 opt
